# crc-exercise-frontend
 
This repository is a setup for the front end of the Cloud Resume Challenge front end using the steps Forrest Brazzeal laid out in the 2021 edition of the Cloud Resume Challenge. Fun fact: Forrest and I crossed paths a couple of jobs ago (for him) when he worked at Trek10 and I worked at one of their clients.

HTML: Had some fun here for my profile photo, as I generated it <a href="https://getavataaars.com">here</a>.

CSS: I used a Bootstrap template from <a href="https://startbootstrap.com/theme/freelancer">here</a> as my starting point. Ended up doing a lot of overhaul but left the core colors the same, so I'm still giving the theme credit.
